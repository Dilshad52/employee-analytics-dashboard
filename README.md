# Employee Performance & Salary Analytics Dashboard

## Project Overview
This project analyzes employee salary, department performance, city-wise employee distribution, and payroll insights using Power BI.

## Tools Used
- Excel
- SQL
- Power BI
- DAX
- Power Query

## Dashboard Features
- KPI Cards
- Salary Analysis
- Employee Distribution
- Salary Trend Analysis
- Interactive Slicers

## Key Insights
- IT department contributes the highest payroll.
- Hyderabad has the highest employee concentration.
- High salary employees are mainly from technical departments.

## Files Included
- Dataset
- SQL Queries
- Power BI Dashboard
- Dashboard Screenshots
- Insights Report
